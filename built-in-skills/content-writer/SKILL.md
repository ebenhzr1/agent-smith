# Content Writer

## Description
Create SEO-optimized, high-quality content following proven copywriting frameworks. Adapts tone, structure, and depth for the target audience and platform.

## Platforms
- Claude Code
- OpenAI Codex
- Gemini CLI
- Generic AI Agent

## Usage

Activate when asked to write articles, blog posts, social media content, landing page copy, or any marketing/content material.

## Process

### Step 1: Understand the brief
- Who is the target audience?
- What is the goal? (educate, convert, entertain, inform)
- What platform? (blog, social, email, landing page)
- What tone? (professional, casual, technical, humorous)
- What SEO keywords should be included?

### Step 2: Research and outline
- Research the topic if needed
- Create a structured outline with:
  - Hook/headline
  - Main points (3-5 for articles)
  - Supporting evidence/data
  - Call to action
- Get outline approval before writing full content

### Step 3: Write with SEO best practices
- Use the target keyword in: title, H1, first paragraph, and naturally throughout
- Write for humans first, search engines second
- Use short paragraphs (2-4 sentences)
- Include subheadings (H2, H3) for scannability
- Add internal/external links where relevant
- Write compelling meta descriptions

### Step 4: Edit and polish
- Remove fluff and redundancy
- Check for readability (aim for grade 8-10 level)
- Verify facts and claims
- Proofread for grammar and spelling
- Format for the target platform

### Step 5: Add call to action
Every piece of content should answer: "What should the reader do next?"
- Blog posts: Subscribe, read related content, comment
- Landing pages: Sign up, buy, learn more
- Emails: Click through, reply, share

## Copywriting Frameworks

| Framework | Use Case | Structure |
|-----------|----------|-----------|
| **PAS** | Problem-oriented content | Problem → Agitate → Solve |
| **AIDA** | Sales copy | Attention → Interest → Desire → Action |
| **4 Cs** | Clear communication | Clear → Concise → Compelling → Credible |
| **FAB** | Feature explanation | Feature → Advantage → Benefit |
| **Story Brand** | Brand messaging | Character → Problem → Guide → Plan → Call to action |

## Anti-Rationalizations

| Excuse | Rebuttal |
|--------|----------|
| "I can skip the outline, I know what to write" | Outlines prevent rambling and ensure structure. Always outline first. |
| "SEO keywords make it sound unnatural" | Good SEO writing is natural. Keywords should flow, not be stuffed. |
| "No one reads long content" | Quality long-form content performs better in SEO and engagement. |

## Verification
- [ ] Target audience is defined
- [ ] Outline created and approved
- [ ] SEO keywords included naturally
- [ ] Writing is scannable (short paragraphs, subheadings)
- [ ] Facts are verified
- [ ] Clear call to action is present
- [ ] Content is proofread
